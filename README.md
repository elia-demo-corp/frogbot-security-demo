# FrogBot Security Demo

Demo repository showcasing JFrog FrogBot capabilities:
- **SCA scanning** - Detects vulnerable dependencies
- **Secrets detection** - Finds hardcoded credentials  
- **Auto-fix PRs** - Automated vulnerability remediation

## Projects

- Root `package.json` - npm dependencies scanned by FrogBot
- `project-one/` - Python application with vulnerable Flask
- `project-two/` - Node.js application source code
- `exposures/` - Files for secrets scanning demo

Managed by JFrog Platform: https://ealonse.jfrog.io
